# projeto-naruto





&#x20;KonohaInfo





Site informativo sobre o universo de Naruto, criado com HTML5 e CSS3.





&#x20;Sobre o projeto





O KonohaInfo é um site desenvolvido como projeto de estudo com o objetivo de praticar estruturação de páginas web, organização de conteúdo e estilização com CSS.





O site apresenta informações sobre personagens, habilidades e principais arcos da história de Naruto.





&#x20;Funcionalidades





\- Página inicial com resumo do universo Naruto

\- Página de história com arcos importantes

\- Página de habilidades do protagonista

\- Navegação entre páginas

\- Layout estilizado com tema inspirado em Naruto

\- Efeitos de hover e transições

\- Responsividade básica para dispositivos móveis





&#x20;Tecnologias utilizadas





\- HTML5

\- CSS3





O que foi aprendido





Durante o desenvolvimento deste projeto foram praticados:





\- Estruturação de sites com múltiplas páginas

\- Organização de CSS com variáveis

\- Uso de classes reutilizáveis

\- Aplicação de imagens via CSS

\- Efeitos de hover e transições

\- Noções de responsividade com media queries





&#x20;Responsividade





O site foi adaptado para dispositivos menores utilizando media queries, ajustando texto, imagens e navegação para melhorar a experiência em telas menores.
